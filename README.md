# ML-rock-paper-scissors

Machine learning application for game Rock-Paper-Scissors

## What is it for?

Main purpose is to make algorithm to classify picture of hand to certain class, which is rock, paper or scissors.

## Used technologies:

```
$ Python
$ joblib
$ matplotlib
$ numpy
$ pandas
$ Pillow
$ scikit-learn
```
